Lista de comandos utilizados:
git init: inicializa um repositório local na
branch main

git config user.name 'Cauã Macêdo' : altera o nome do usuário local do projeto

git config user.email 'caua.macedo.n@gmail.com' : altera o e-mail do usuário local do projeto

git add . : adiciona todos os arquivos ao futuro commit

git commit -m “primeiro commit”:realiza um commit

Comandos básicos (após criar o projeto no site):
• git remote add origin-github https://github.com/mac-edx/curso-git-embarcatech.git: adiciona o repositório remoto

• git push -u origin-gitHub  main : envia o conteúdo da branch main para
o repositório remoto.
